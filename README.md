# Peace Designs — Official Portfolio Website

**Creativity Without Limits**

The official portfolio and business website for **Peace Designs**, 
a full-service graphic design studio founded by **Shaloam Dixon**, 
based in Kingston, Jamaica.

## About This Site

This is a single-file, self-contained HTML portfolio website built 
to showcase graphic design work to clients locally and internationally. 
All images are embedded directly in the file — no external dependencies 
or internet connection required to view it.

## What's Inside

- Hero section with studio branding and stats
- About section — the designer behind the work
- Portfolio gallery — 22 works across Church, Events, 
  Business, Social & Personal categories with filter system
- Services section — 6 design service offerings
- Creative Products Store — purchasable design packages
- Client Testimonials
- Contact & Inquiry Form
- Fully responsive — works on mobile, tablet and desktop
